The "Steamhours" website has been made, to provide people using Games Distribution platform "Steam", with information about their games (playtime,statistics and so on) on base of Steam WebAPI.

It was created using JavaScript,jQuery, PHP7 and CSS3+HTML5.
My portfolio was my first project after learning the basics of Web Development.

Technologies used in creation of it are: HTML5,CSS3 and Bootstrap Framework.
